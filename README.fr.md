# 🥋 BJJ Dojo

**BJJ Dojo** est votre compagnon d'entraînement de Jiu-Jitsu Brésilien offline-first, conçu pour vous aider à vous entraîner avec intention, suivre vos progrès réels et organiser votre jeu.

## 🚀 Commencez maintenant (sans configuration)

Vous pouvez utiliser l'application immédiatement dans votre navigateur :

👉 **https://samoca95.github.io/bjj-dojo/**

Aucun compte requis. Aucun backend requis.

## 📲 Installez-la sur votre téléphone (comme une app native)

BJJ Dojo est une PWA : vous pouvez l'installer depuis le navigateur et l'utiliser comme n'importe quelle autre application.

### Android (Chrome)

1. Ouvrez **https://samoca95.github.io/bjj-dojo/**
2. Appuyez sur le menu du navigateur (**⋮**)
3. Appuyez sur **Installer l'application** / **Ajouter à l'écran d'accueil**
4. Confirmez l'installation

### iPhone (Safari)

1. Ouvrez **https://samoca95.github.io/bjj-dojo/**
2. Appuyez sur **Partager** (carré avec flèche)
3. Appuyez sur **Sur l'écran d'accueil**
4. Appuyez sur **Ajouter**

Une fois installée, ouvrez BJJ Dojo depuis votre écran d'accueil comme n'importe quelle autre app.

## ✨ Pourquoi BJJ Dojo

- 🎯 **Entraînez-vous avec clarté** : notez ce qui s'est passé sur le tatami, pas seulement la durée.
- 📈 **Suivez vos progrès** : conservez un historique cohérent des sessions et des techniques.
- 🔒 **Vos données vous appartiennent** : elles restent dans votre navigateur et fonctionnent hors ligne.
- 🧩 **Personnalisez l'app** : adaptez clubs, catégories et disposition de l'accueil.

## 🛠️ Fonctionnalités principales

### 📝 Suivi des sessions

- Enregistrez date, durée, type de session, énergie, notes, taps et techniques.
- Organisez vos clubs et sélectionnez le bon club pour chaque session.

### 📚 Bibliothèque de techniques

- Parcourez et recherchez des techniques rapidement.
- Filtrez par catégorie et difficulté.
- Créez des techniques personnalisées et reliez enchaînements/contres.

### ⚙️ Personnalisation

- Définissez ceinture et barrettes.
- Personnalisez les icônes de catégories (emoji ou icônes intégrées).
- Réorganisez les sections d'accueil et fixez des objectifs hebdomadaires de temps sur le tatami.
- Changez le thème et la langue dans les paramètres.

### 💾 Sauvegarde & restauration

- Exportez vos données en JSON.
- Importez une sauvegarde pour restaurer votre progression sur le même appareil ou un autre.

### 📡 Expérience offline-first

- Fonctionne sans internet après le chargement initial.
- Parfait pour les salles avec connexion instable.

## 🔐 Données & confidentialité

Toutes les données sont stockées localement dans votre navigateur (IndexedDB). Il n'y a pas de compte backend ni de profil côté serveur.
